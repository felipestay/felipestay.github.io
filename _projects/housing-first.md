**Vivienda Primero** is Chile's Housing First program, implemented by Corporación Moviliza in Santiago and Valparaíso since 2019. This project involves ongoing program evaluation including retention rate analysis, fidelity assessment, and longitudinal outcome tracking.

### Key findings
- Retention rates above 80% across program cohorts
- Fidelity assessment using validated Housing First Fidelity Scale
- Administrative data from 100+ program participants (2019–2024)

### Output
- OSF Preprint: [Evaluation of the Fidelity of the Housing First Program in Chile](https://doi.org/10.17605/OSF.IO/HM7CY)
